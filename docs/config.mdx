# Mod Configuration

The `biomeblends-common.toml` config allows global changes to blend behavior, which can be useful for both players and modpack creators. This can be useful for simpler changes without the need to edit or create data packs.

## Blend Radius Scale

The blend radius can be modified using the following two config options: 

- `horizontal_scale` 
- `vertical_scale` 

These act as **multipliers** on the radius values defined in the data pack. They preserve the relative differences between blend types (e.g. larger blends stay larger than smaller ones).

Blend Scale supports values from 0.0 to 10.0. For example, setting `horizontal_scale` to `0.5` will cut the area of effect of all blends by half.

## Blend Source in Tooltip

Displays blend's source in its tooltip. If the specified source is a currently loaded mod, its display name is automatically used instead of the raw mod ID.

This can be useful when multiple mods add blends with similar names, making it easy to tell them apart.

## Ignore Vertical Radius

`ignore_vertical_radius` can be set to `true` to apply all blends from **the bottom to the top of the world**, ignoring any vertical radius settings.

## Mod Compatibility

This section can be used to disable compatibility datapacks for different mods. Please note that for compatibility datapacks to work in the first place, the matching mod needs to be loaded.
